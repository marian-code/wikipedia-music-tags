Welcome to Wiki music's documentation!
======================================

Documentation is still under construction !!! but most parts are commplete.

Projects home directory is:
`wikipedia-music-tags <https://github.com/marian-code/wikipedia-music-tags>`_. 

Wiki music is a music tagger written in pure python. It has the ability to
parse Wikipedia for music tags, download lyrics, coverart and writing it all
into music file tags. It has two use modes one as a CLI tool which lacks some
features, coverart manipulation and the fully featured GUI version written in
Qt which can use both PyQt and PySide python backends.
